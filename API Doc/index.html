Introdução

Todo API Doc foi construído através de APIs públicas onde você pode consumir elas da forma que desejar (respeitando as políticas de uso).

A comunicação é feita através de HTTPS usando GET ou POST. Tanto a solicitação quanto a resposta são formatadas como JSON e o tipo de conteúdo de ambas é application/json.

Base URL

Todas as URLs incluídas nessa documentação exigem a BaseUrl!

                    http://localhost:2500

Autenticação cliente

Criar cliente

A criação de um cliente é feita automáticamente assim que ele manda um arquivo para ser impresso.

Parâmetro Tipo Required Descrição
Email String True O email informado pelo cliente.
Nome String False O nome informado pelo cliente.
Telefone String False O número informado pelo cliente.

⚠️ Não há a nescecidade de criar um cliente de outra maneira. ⚠️

Buscar todos clientes

Para fazer um requisição, onde ela irá retornar todos os clientes em nossa base de dados, basta fazer esse GET:

                    GET {{BaseURL}}/clientes

Buscar um cliente

Para fazer um requisição, onde ela irá retornar apenas o cliente informado, basta fazer esse GET:

                    GET {{BaseURL}}/clientes/{idCliente}

Editar cliente

Para enviar uma solicitação, onde ela irá alterar os dados do cliente, basta fazer esse PUT:

                    PUT {{BaseURL}}/clientes/{idCliente}

Excluir cliente

Para enviar uma solicitação, onde ela irá excluir os dados do cliente, basta fazer esse DELETE:

                    DELETE {{BaseURL}}/clientes/{idCliente}

Autenticação arquivo

Criar arquivo

Para enviar uma solicitação, onde ela irá criar um arquvivo, basta fazer esse POST:

                        POST {{BaseURL}}/arquivos

Buscar todos arquivos

Para fazer um requisição, onde ela irá retornar todos os arquivos, basta fazer esse GET:

                        GET {{BaseURL}}/arquivos

Buscar um arquivo

Para fazer um requisição, onde ela irá retornar apenas o arquivo informado, basta fazer esse GET:

                        GET {{BaseURL}}/arquivos/{idArquivo}

Atualizar um arquivo

Para enviar uma solicitação, onde ela irá atualizar um arquivo, basta fazer esse POST:

                        PUT {{BaseURL}}/arquivos/{idArquivo}

Deletar um arquivo

Para enviar uma solicitação, onde ela irá deletar um arquvivo, basta fazer esse DELETE:

                        DELETE {{BaseURL}}/arquivos/{idArquivo}
